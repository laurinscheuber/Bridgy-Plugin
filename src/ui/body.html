settings_suggest Commit vs. Merge Request Strategies expand_more

Merge Requests (Recommended): Committing edits will generate a new branch and MR automatically. Safe for collaboration.

Direct Commits: Instantly overwrites tracked files on the target branch. Faster for mature component libraries.

Reset Configuration

⚠️ Warning: This action cannot be undone!

This will permanently delete all configuration data for this Figma project, including:

  • GitLab Project ID and access token
  • File path and branch settings
  • Strategy preferences (commit vs merge request)
  • All saved metadata (who saved, when)

After reset, you'll need to reconfigure everything from scratch.
